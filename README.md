This Python project was created with [Dash](https://dash.plotly.com/) framework.
## Getting Started
First of all, you will need to install Python 3.9.5+ if you don't have one.

Then download project to folder and install all libraries using command:
```
pip install -r requirements.txt
```
Run `my.py` file and open http://127.0.0.1:8050/ with your browser to start working with the graph editor.
